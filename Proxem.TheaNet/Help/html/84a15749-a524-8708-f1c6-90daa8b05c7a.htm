Function_R(IEnumerableIVar, IExprR, OrderedDictionary, IDictionary)
Creates a function with one output. The number of inputs and their types are determined during compilation. At runtime the function will fail if it don't receives as many inputs than expected. The inputs will be downcasted to the expected type inside the function.
