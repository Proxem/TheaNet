Reshape(Int32, Boolean, Boolean)
Reshapes the array to the given shape. The reshape operation tries to be a O(1) and to reuse the underlying values. If it fails to do so (for instance if this a transposed array), Reshape will either throw an exception or copy the value in a new structure. (Note: like in NumPy the default is allowCopy=true). If one of the shape value is -1, will try to guess the correct value.
