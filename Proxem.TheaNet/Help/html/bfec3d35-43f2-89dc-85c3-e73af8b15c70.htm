Remarks
The generic case of convolution requiers Jacobian computation. As this not implemented in Theanet, the user as to provide himself the "Jacobian". This operator make the following assumptions: if: z = ConvolveCustom(x, y, f) then: dx = CorrelateCustom(dz, y, g) dy = CorrelateCustom(dz, x, h)
