LogisticRegression Class
Multi-class Logistic Regression Class http://deeplearning.net/tutorial/logreg.html The logistic regression is fully described by a weight matrix :math:`W` and bias vector :math:`b`. Classification is done by projecting data points onto a set of hyperplanes, the distance to which is used to determine a class membership probability.
